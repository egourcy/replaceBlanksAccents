# replaceBlanksAccents
Replace blank spaces with "-" character &amp; remove accents from file name

Built in Python 3.11.3:

## Requirements

- Python 3.x

## Example

### Windows

py.exe removeBlanks.py c:\path\to\folder

### GNU/Linux

python3 removeBlanks.py /path/to/folder

#### Result

file-name-out.txt
